# Password Security Toolkit

A simple toolkit built in Python that consists of tools to evaluate and generate passwords.

## Features
- Check the strength of a password — `Strong`, `Medium`, or `Weak`
- Generate a password based on conditions entered by the user

## Tech Stack
- Language: `Python`
- Libraries: `string`, `random`

## How to Run
```bash
python main.py
```

## Usage

### 1. Password Strength Checker
- Enter a password
- Checks if the password is at least 8 characters long, and contains at least one number, one lower-case letter, one upper-case letter, and one special character
- Displays the strength of the password based on how many conditions it satisfies
- If the password is `Weak`, the user is offered the option to use the `Random Password Generator` tool instead

### 2. Random Password Generator
- The user enters the length and the character types the password should include (upper-case, numbers, special characters)
- The tool generates a random password satisfying those conditions
- The user can then use the `Password Strength Checker` tool to check the strength of the generated password

### 3. Exit
- Exits the program

## How It Works
- Password strength is scored by checking 5 conditions: length, upper-case, lower-case, numbers, and special characters — the number of conditions met determines whether the password is Strong, Medium, or Weak
- The checker also compares the entered password against a list of common weak passwords (e.g. `password123`, `qwerty`) and flags them as too common regardless of the score
- The generator builds a character pool starting with lower-case letters by default, then adds upper-case letters, digits, or special characters based on what the user chooses, using `string.ascii_uppercase`, `string.digits`, and `string.punctuation`
- A password is generated by randomly picking characters from this pool using `random.choice()`, one character at a time, for the length specified

## Output
- All output is displayed directly in the terminal
- The strength checker prints the strength level along with specific suggestions for any missing conditions (e.g. "Password should contain at least 1 number")
- The generator prints the final generated password

## What I Learned
- Using the `string` module's constants (`string.ascii_lowercase`, `string.ascii_uppercase`, `string.digits`, `string.punctuation`) to build a character pool for password generation
- Using `random.choice()` to randomly sample characters from the pool
- Using `sum()` on a list of boolean conditions to score password strength
- Using default parameter values in functions (e.g. `offer_generator=True`) to control optional behavior
